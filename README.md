# PVPARENA Site

PvPArena 개인정보처리방침 웹사이트입니다.
한국어/영어 전환을 지원하며, Epic Online Services(EOS) 사용에 따른 개인정보 처리 안내를 제공합니다.

## 기술 스택

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Lucide React (아이콘)

## 시작하기

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
npm run preview
```
